### Usage

##### Import the component:

<Source
  language="javascript"
  code={dedent`import { ScrollableList } from '@kws3/ui';`}
/>

<div class="columns">
<div class="column">
<p>Normal Usage Example:
<br/>You can define the <code>let</code> variables on main component</p>

```jsx
<ScrollableList
  {items}
  {height}
  {item_height}
  {end_threshold}
  {padding_threshold}
  bind:start
  bind:end
  on:end={endReached}
  let:item
  let:index
>
  <ListItem {item} {index} />
</ScrollableList>
```

</div>
<div class="column">
<p>Usage with loader slot Example:
<br/>You have to define the <code>let</code> variables in a slot wrapper</p>

```jsx
<ScrollableList
  {items}
  {height}
  {item_height}
  {end_threshold}
  {padding_threshold}
  bind:start
  bind:end
  on:end={endReached}
>
  <span
    slot="default"
    let:item
    let:index
  >
    <ListItem {item} {index} />
  </span>
  <span slot="loader">Loading...</span>
</ScrollableList>
```

</div>
</div>
