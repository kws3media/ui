# Divider
## Component to display a horizontal or vertical divider with optional text.


<Meta of={Stories} />
<Canvas />
<Controls />


### Usage

##### Import the component:
<Source
  language='javascript'
  code={ dedent`import { Divider } from '@kws3/ui';`}
/>

<div class="notification is-light is-info">Use <code>m-*</code> helper classes from Bulma to precisely control the spacing around the divider.</div>


##### Import the CSS:
<Source
  language='javascript'
  code={ dedent`@import '@kws3/ui/styles/Divider.scss';`}
/>

##### SCSS customization variables with default values:
<Source
  language='css'
  code={ dedent`
    $kws-theme-colors: $colors;
    $kws-divider-color: $border;
    $kws-divider-font-size: $size-7;
    $kws-divider-gap: 0.25em;
    $kws-divider-margin: 1.25rem;
  `}
/>
