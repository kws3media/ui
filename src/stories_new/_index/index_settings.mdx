## Framework Settings
Framework wide settings can be applied in one place.
Ideally in the entry file of your app.

### Usage
```js
import {applySettings} from '@kws3/ui';

applySettings({
  defaultIconFamily: 'fa',
  defaultToastPlacement: 'bottom',
  defaultSnackbarPlacement:'bottom-right',
  defaultNotificationPlacement:'top-right'
  hasTransitions: true,
  defaultChartColors: [
    "#284BED",
    "#ED6134",
    "#1DAFEC",
    "#EDB405",
    "#11EDB7",
    "#77ED11",
  ]
});

//you can also apply just one setting:
applySettings({
  hasTransitions: false
});
```
