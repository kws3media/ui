### capitaliseFirstLetter

The `capitaliseFirstLetter` function converts the first character of a string to uppercase.

**Parameter:**
`string`

**Example:**

```js
import { capitaliseFirstLetter } from "@kws3/ui/utils";
capitaliseFirstLetter(string);
```
