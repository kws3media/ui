## randomPercent

The `randomPercent` returns a random percentage.

**Parameters:** 

1. min : `type: Number`, Default `1`
2. max : `type: Number`, Default `100`

#### Usage

```js
import { randomPercent } from "@kws3/ui/utils";
randomPercent(min, max);
randomPercent(min);
randomPercent();
```
