### Usage
#### Import the component:
<Source
  language="javascript"
  code={
    dedent`import { Timepicker } from '@kws3/ui';`
  }
/>

#### Import the SCSS:
<div class="notification is-warning"><strong>Note:</strong> <em>Need to import flatpicker SCSS file first</em></div>
<Source
  language="javascript"
  code={
    dedent`
    @import 'flatpickr/dist/flatpickr';
    @import '@kws3/ui/styles/Timepicker.scss';`
  }
/>

#### SCSS customization variables with default values:
<Source
  language="css"
  code={
    dedent`
      $kws-theme-colors: $colors;
      $kws-timepicker-background: $background;
      $kws-timepicker-text: $text;
    `
  }
/>
