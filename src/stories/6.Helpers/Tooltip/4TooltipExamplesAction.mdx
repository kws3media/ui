### Tooltip Action
If you do not have a lot of tooltips in your app, you can use the `tooltip` action to sprinkle tooltips on selected nodes.

<div className="notification is-warning">
<p className="sb-unstyled">Tooltip Action does not use event delegation. <br/>Therefore, it can affect performance if there are a large number of tooltips.<br/><br/>The recommended way is to use tooltips via event delegation, see: <a href="./?path=/docs/helpers-tooltips-1-overview--docs">Tooltip Overview</a></p>
</div>



<div className="columns">
  <div className="column">

The action accepts an object of parameters.

For a list of other supported parameters, see:  https://atomiks.github.io/tippyjs/v6/all-props/

By default, it tries to use the content of `data-tooltip` attribute as the tooltip content.
However, you can also send a string as a parameter to the action.
And that will be used instead of the attribute value.

  </div>
  <div className="column">

```html
<!--send content as parameter-->
<div use:tooltip={{content: "Some Text"}}>
  Some Content
</div>

<!--send content as data-tooltip attribute-->
<div
  use:tooltip
  data-tooltip="Some Text">
  Some Content
</div>

<script>
  import { tooltip } from '@kws3/ui'; //notice lowercase import name
</script>
```

  </div>
</div>

Parameters can be sent during action initialisation, or as `data-tippy-*` attributes.
The following examples are functionally indentical:

<div className="columns">
  <div className="column">

```html
<div
  data-tooltip="Some text in tooltip"
  use:tooltip={{
    placement: 'bottom',
    offset: [0, 10]
  }}>
  Some Content
</div>
```

  </div>
  <div className="column">

```html
<div
  data-tooltip="Some text in tooltip"
  use:tooltip
  data-tippy-placement="bottom"
  data-tippy-offset="[0, 10]"
  >
  Some Content
</div>
```

  </div>
</div>
