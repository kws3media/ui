# Divider
## Component to display a horizontal or vertical divider with optional text.


<Meta of={Stories} />
<Canvas />
<Controls />


### Usage

##### Import the component:
<Source
  code={ dedent`import { Divider } from '@kws3/ui';`}
/>

<div className="notification is-light is-info">Use <code>m-*</code> helper classes from Bulma to precisely control the spacing around the divider.</div>


##### Import the CSS:
<Source
  code={ dedent`@use '@kws3/ui/styles/Divider.scss';`}
/>

##### SCSS customization variables with default values:
<Source
  code={ dedent`
    @use '@kws3/ui/styles/Divider.scss' with (
      $theme-colors: dv.$colors !default,
      $color: cv.getVar("border") !default,
      $font-size: cv.getVar("size-smal") !default,
      $gap: 0.25rem !default,
      $margin: 1rem !default
    );
  `}
/>
