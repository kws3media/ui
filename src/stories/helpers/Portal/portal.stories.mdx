### Usage

##### Import the component:
<Source
  language='javascript'
  code={ dedent`
  <Portal target="body">
    <div class="toast">Entity successfully updated!</div>
  </Portal>
  
  <script>
    import { Portal } from '@kws3/ui';
  </script>
  `}
/>

##### Import the action:
<Source
  language='javascript'
  code={ dedent`
  <div class="toast" use:portal={document.body}>Entity successfully updated!</div>
  
  <script>
    import { portal } from '@kws3/ui';
    // notice the character case
  </script>
  `}
/>
