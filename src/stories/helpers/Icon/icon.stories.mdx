### Usage
#### Requires FontAwesome version <= 4.7.0, this can be either loaded from your bundle, or from a CDN
#### For Unicons icons, requires css loading as mentioned in https://iconscout.com/unicons this can be either loaded from your bundle, or from a CDN
#### For GG icons, requires css loading as mentioned in https://css.gg/ this can be either loaded from your bundle, or from a CDN
#### For Line icons, requires css loading as mentioned in https://icons8.com/line-awesome/howto this can be either loaded from your bundle, or from a CDN
