### fontawesome.io
#### Requires FontAwesome version <= 4.7.0, this can be either loaded from your bundle, or from a CDN
<Source
  language='html'
  code={ dedent`<Icon icon="check" color="" size="" />`}
/>

### iconscout.com/unicons
#### For Unicons icons, requires css loading as mentioned in https://iconscout.com/unicons this can be either loaded from your bundle, or from a CDN
<Source
  language='html'
  code={ dedent`<Icon family="unicons" icon="check" color="" size="" />`}
/>

### css.gg
#### For GG icons, requires css loading as mentioned in https://css.gg/ this can be either loaded from your bundle, or from a CDN
<Source
  language='html'
  code={ dedent`<Icon family="gg" icon="check" color="" size="" />`}
/>

### icons8.com/line-awesome
#### For LineAwesome icons, requires css loading as mentioned in https://icons8.com/line-awesome/howto this can be either loaded from your bundle, or from a CDN
##### Icon can have 3 styles: regular, solid or brand. For each style can use `lar` `las` or `lab` accordingly.
<Source
  language='html'
  code={ dedent`<Icon family="lar" icon="check" color="" size="" />
  <Icon family="las" icon="check" color="" size="" />`}
/>

### Usage
#### Import the component:
<Source
  language='javascript'
  code={ dedent`import { Icon } from '@kws3/ui';`}
/>

#### Set default icon type:
Instead of specifying an icon family on each icon, it can be set globally on your app, using the static method available
<Source
  language='javascript'
  code={ dedent`import { Icon } from '@kws3/ui';
  Icon.setDefaultIconType("lar");
  //now all icons will use this family unless overriden on a per icon basis`}
/>
