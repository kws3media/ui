Notification

A Notification component for showing notifications.

Basic example

Your have 41 notifications

 
<Notification>
  <p>Your have 41 notifications.</p>
</Notification>

Type

Your have 41 notifications

 
// type : info (default), primary, success, warning, danger
<Notification type='danger'>
  <p>Your have 41 notifications.</p>
</Notification>

Dismissable

Your have 41 notifications. And you can dismiss this notification.

 
<Notification dismissable={true}>
  <p>Your have 41 notifications. And you can dismiss this notification.</p>
</Notification>

Usage

Import the component:

import { Notification } from '@kws3/helpers';
