Notification

A Notification component for showing notifications.

Example

Use Knobs to check out all available options on this component

Your have 41 notifications

 
<Notification color="{color}">
  <p>Your have 41 notifications.</p>
</Notification>

Dismissable

You have 41 notifications. And you can dismiss this notification.

 
<Notification dismissable={true}>
  <p>Your have 41 notifications. And you can dismiss this notification.</p>
</Notification>

Usage

Import the component:

import { Notification } from '@kws3/helpers';
