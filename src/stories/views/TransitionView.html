As can be seen from code examples above, some transition types accept extra properties relating to the transition. For example, fly accepts x, and scale accepts from.

In addition to the properties in the examples. All transitions also accept the following properties:
  • - delay - in milliseconds, delays the start of the transition
  • - classes - CSS classes to to add to the wrapper element.
  • - transition - boolean, if false, transition will not run, and behave as if its a static element.

Usage

Import the component:

{ `import Transition from '@kws3/transitions'; `}

Import the SCSS:

@import 'path/to/node_modules/@kws3/transitions/styles';
