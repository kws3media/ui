Events Fired:
  • - close: When closed via modal close button
  • - clickOutside: When overlay is clicked outside modal
Methods:
  • - close(): example: this.refs.my_modal.close()
  • - open(): example: this.refs.my_modal.open()

Usage

Import the component:

{ `import { Modal, CardModal } from '@kws3/helpers'; `}

Import the SCSS:

@import 'path/to/node_modules/@kws3/helpers/components/Modal/styles.scss';
