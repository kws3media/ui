Loader

A Loader component for display FontAwsome icon.

Basic example


<Loader/>

Color


// transparent (default), info, danger, warning, primary 

 <Loader color="info"/>

Large Loader


 <Loader large_loader={true}/>

Usage

Import the component:

import { Loader } from '@kws3/helpers';

Import the SCSS:

@import 'path/to/node_modules/@kws3/helpers/components/Loader/styles';

SCSS customization variables with default values:

$kws-loader-width: 2rem;
$kws-loader-height: 2rem;
$kws-loader-width-large: ($kws-loader-width * 3);
$kws-loader-height-large: ($kws-loader-height * 3);
$kws-loader-color: #ccc;
$kws-loader-overlay-background: rgba(0,0,0,0.15);
