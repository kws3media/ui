Loader

Graphic to display a loading status to users.

Example

Use Knobs to check out all available options on this component

<Loader
  color="{color}"
  size="{size}"
  type="{type}"
  large_loader="{{ large_loader ? 'true' : 'false' }}"
/>
  • Note 1: In addition to the above options, the component also accepts a style property. This lets you add custom css to the component.
  • Note 2: When this component is inside a div with class .is-overlay, only the large_loader property is respected, all other properties are ignored.

Usage

Import the component:

import { Loader } from '@kws3/helpers';

Import the SCSS:

@import 'path/to/node_modules/@kws3/helpers/components/Loader/styles';

SCSS customization variables with default values:

$kws-loader-width: 2rem;
$kws-loader-height: 2rem;
$kws-loader-width-large: ($kws-loader-width * 3);
$kws-loader-height-large: ($kws-loader-height * 3);
$kws-loader-color: #ccc;
$kws-loader-overlay-background: rgba(0,0,0,0.15);
