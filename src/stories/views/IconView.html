In addition to the above options, the component also accepts the following properties:

  • icon_class - to add a custom class
  • fa_class - to add a custom FA class, like fa-spin, fa-rotate-90 etc
  • fa_style - to add custom raw css to the inner fontawesome <i>

Usage

Requires FontAwesome version <= 4.7.0,
this can be either loaded from your bundle, or from a CDN
For GG icons, requires css loading as mentioned in https://css.gg/
this can be either loaded from your bundle, or from a CDN

{ `import { Icon } from '@kws3/helpers'; `}
