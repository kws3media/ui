ConfirmButton

Multi-purpose Button with built-in confirmation.

Example

Use Knobs to check out all available options on this component

<ConfirmButton
  text="Confirm"
  color="{color}"
  size="{size}"
  icon_only="{{icon_only ? 'true' : 'false'}}"
/>

Usage

Import the component:

import { ConfirmButton } from '@kws3/buttons';
