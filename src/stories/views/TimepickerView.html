A Timepicker component to pick time.

Example

Pick a time

<Timepicker bind:value="new_time" />

Usage

Requires flatpickr version 3.0.6,
 
npm install flatpickr --save

Import flatpickr style file to app.scss

@import "path/to/node_modules/flatpickr/dist/flatpickr";

Import the component:

import Timepicker from '@kws3/timepicker';

Import the SCSS:

@import 'path/to/node_modules/@kws3/timepicker/styles';
$kws-timepicker-boder-radius: 3px 3px !default;
$kws-timepicker-border: 1px solid $primary !default;
$kws-timepicker-am-pm-color: $primary !default;
