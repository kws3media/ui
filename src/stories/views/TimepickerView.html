Usage

Import the component:

import Timepicker from '@kws3/timepicker'; Requires flatpickr SCSS to be loaded first

Import flatpickr style file to app.scss

@import "path/to/node_modules/flatpickr/dist/flatpickr";

Import the SCSS:

@import 'path/to/node_modules/@kws3/timepicker/styles'; $kws-timepicker-border-radius: 3px 3px !default; $kws-timepicker-border: 1px solid $primary !default; $kws-timepicker-am-pm-color: $primary !default;
