Icons in checkbox

By default it uses the check icon. But any icon can be sent. Also, an empty value can be sent as an icon, and it will render the tick mark using html.

Agree Agree Agree Agree { `icon=""`}
Agree Agree Agree Agree { `icon="angle-double-right"`}
Agree Agree Agree Agree { `icon="chevron-right"`}
Agree Agree Agree Agree { `icon="arrow-right"`}

Usage

Import the component:

{ `import { Checkbox } from '@kws3/controls'; `}

Import the SCSS:

@import 'path/to/node_modules/@kws3/controls/components/checkbox/styles.scss';

SCSS customization variables with default values:

$kws-checkbox-default-check-color: #333 !default; $kws-checkbox-default-check-color-invert: findColorInvert($kws-checkbox-default-check-color) !default; $kws-checkbox-default-border-color: #dbdbdb !default;
