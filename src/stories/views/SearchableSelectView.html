Basic example



<SearchableSelect data="{brands}" placeholder="Choose a Brand..." bind:value="selected_brand" searchKey="name" searchValue="id" />

Here brands is an array of this format:

[ { id: 1, name: 'LG'}, { id: 2, name: 'Samsung'}, { id: 3, name: 'Xiaomi'}, ... ]

searchKey tells the component which key to search in, as you type.
In the example above, we want to search the name key.

searchValue is the keys whose value will be set as the value of the select element once a corresponding entry has been selected.
In the example above, we want the value set to 1 when we select Apple, 2 when we select HTC etc.

Usage

Import the component:

import SearchableSelect from '@kws3/searchable-select';

Import the SCSS:

@import 'path/to/node_modules/@kws3/searchable-select/styles';

SCSS customization variables with default values:

$kws-ss-item-selected-bg-color:#0071bc; $kws-ss-item-selected-text-color:#ffffff; $kws-ss-item-selecting-bg-color:#0071bc; $kws-ss-item-selecting-text-color:#ffffff; $kws-ss-dropdown-bg-color: #ffffff; $kws-ss-dropdown-text-color: #333; $kws-ss-caret-color: #0071bc;
