Datepicker

A Datepicker component to pick date.

Example

Pick a date

<Datepicker bind:value="new_date" />

Usage

Import the component:

import Datepicker from '@kws3/datepicker';
Requires flatpickr SCSS to be loaded first

Import flatpickr style file to app.scss

@import "path/to/node_modules/flatpickr/dist/flatpickr";

Import the component SCSS:

@import 'path/to/node_modules/@kws3/datepicker/styles';

SCSS customization variables with default values:

$kws-datepicker-arrow-top-color: $primary;
$kws-datepicker-arrow-bottom-color: $primary;

$kws-datepicker-month-color: $white;
$kws-datepicker-month-bg-color: $primary;
$kws-datepicker-month-fill-color: $white;
$kws-datepicker-month-boder-radius: 3px 3px 0 0;

$kws-datepicker-weekdays-bg-color: $primary;

$kws-datepicker-weekday-bg-color: $primary;
$kws-datepicker-weekday-color: $primary-invert;

$kws-datepicker-prev-next-hover-color: $dark;
$kws-datepicker-prev-next-hover-svg-color: $dark;

$kws-datepicker-days-border-radius: 0 0 3px 3px;
$kws-datepicker-left-border: 1px solid $primary;
$kws-datepicker-right-border: 1px solid $primary;
