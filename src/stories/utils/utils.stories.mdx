## 1. truncate

The truncate() function shorten the given String to a specified length.

#### Usage

```js
import { truncate } from "@kws3/ui/utils";
truncate(string, length);
```

## 2. nl2br

The nl2br function replace each newline (`\n`) with HTML line breaks (`<br>`) in a string.

#### Usage

```js
import { nl2br } from "@kws3/ui/utils";
nl2br(input);
```

## 3. cloneObject

The cloneObject function used to clone js object.

#### Usage

```js
import { cloneObject } from "@kws3/ui/utils";
cloneObject(obj);
```

## 4. rAF

The rAF function - Request an animation before the next repaint.

#### Usage

```js
import { rAF } from "@kws3/ui/utils";
rAF(callBackFunction);
```

## 5. debounce

The function that you want to execute after the debounce time.

#### Usage

```js
import { debounce } from "@kws3/ui/utils";
debounce(callBackFunction, threshold);
```

## 6. capitaliseFirstLetter

The `capitaliseFirstLetter` function converts the first character of a string to uppercase.

#### Usage

```js
import { capitaliseFirstLetter } from "@kws3/ui/utils";
capitaliseFirstLetter(string);
```

## 7. createDate

The createDate function makes mysql dates work in safari.

#### Usage

```js
import { createDate } from "@kws3/ui/utils";
createDate(date);
```

## 8. Smart Forms

The `makeForm`, `makeForms` helpers used for tracking form data is dirty or not.

#### Usage

```js
import { makeForm, makeForms } from "@kws3/ui/form";
makeForm(data);
```

## 9. Resize Observer

The `resizeObserver` used to handle window resizing.

#### Usage

```js
import { hasResizeObserver, resizeObserver } from "@kws3/ui/resizeObserver";
if(hasResizeObserver){
  <div use:resizeObserver on:resize={resizeHandler}>
}
```

## 10. fuzzysearch

The `fuzzysearch` used to filter the object.

#### Usage

```js
import { fuzzysearch } from "@kws3/ui/search";
fuzzysearch(needle, haystack, opts);
```

## 11. keyboardEvents

The `keyboardEvents` used to trigger the events based on the key stroke.

#### Usage

```js
import { keyboardEvents } from "@kws3/ui/keyboard";
keyboardEvents(node);
```
