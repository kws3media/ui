### rAF

The `rAF` method tells the browser that you wish to perform an animation and requests that the browser calls a specified function to update an animation before the next repaint. The method takes a callback as an argument to be invoked before the repaint.

**Parameter:**
`call back function`

```js
<div id="animatedDiv" />;
import { rAF } from "@kws3/ui";

let globalID;
function repeatOften() {
  var div = document.getElementById("animatedDiv");
  div.innerHTML += `<div class="orange"></div>`;
  globalID = rAF(repeatOften);
}
globalID = rAF(repeatOften);

function start() {
  globalID = rAF(repeatOften);
}

function stop() {
  cancelAnimationFrame(globalID);
}
```
