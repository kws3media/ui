### fileDownloader

Function used to clone js object to another.

**Parameter:**
data: `type : object`
fileName: `type : string`

```js
import { fileDownloader } from "@kws3/ui/utils";
let obj = [{ name: "User", surname: "A" }, { name: "User", surname: "B" }];

$: fileDownloader(obj, "users.csv");
```
