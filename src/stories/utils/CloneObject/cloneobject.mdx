### CloneObject

The `cloneObject` function is used to clone any js object.

**Parameter:**
`object`

```js
import { cloneObject } from "@kws3/ui/utils";
let obj = {
  title: "Mr",
  name: "User",
};

$: clone = cloneObject(obj);
```
