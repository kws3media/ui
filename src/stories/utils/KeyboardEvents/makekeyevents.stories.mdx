### makeKeyDefinition

The `makeKeyDefinition` is a factory helper used to create custom keydown handler events. 
Key combinations can be made with any modifier keys from `Shift`, `Ctrl`, `Alt`, `Meta` with characters, numbers or function keys.

#### **Parameters**
- definition : `type: Number | string` can be a string like `Enter`, `Tab` or number as keyCode, also allows key combinations like `ctrl+d`, `ctrl+alt+x`
- CommandKey : `type: boolean` if true, in mac 'ctrl' key binding will be shift on 'command' key.

**Returns:**
Function take params, node :`type : DOM Element` and event : `type: function`.

#### Usage

```js
import {makeKeyDefinition} from "@kws3/ui/keyboard";
const save = makeKeyDefinition(keyCombination) 
```

**Example:**
```html
<input type="text" use:ctrlAltC={(event) => {eventKeyPressed(event, "ctrl+alt+c")}} />

<script>
  import {makeKeyDefinition} from "@kws3/ui/keyboard";

  const ctrlAltC = makeKeyDefinition("ctrl+alt+c");

  function eventKeyPressed(event, key) {
    eventText = key + " Pressed";
  }
</script>
```
