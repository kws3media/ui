### makeKeyDefinition

The `makeKeyDefinition` is factory helper to create custom keydownHandler events.

**Parameters:**
code : `type: Number` Key code

**Returns:**
Function take params, node :`type : DOM Element` and event : `type: function`.

#### Usage

```js
import {makeKeyDefinition} from "@kws3/ui/keyboard";
const save = makeKeyDefinition(keyCode) 
```

**Example:**
```html
<input type="text" use:save={saveEvent} />
<script>
  import {makeKeyDefinition} from "@kws3/ui/keyboard";

  let opts = {
    code: 17, //ctrl key code
    key: "S",
  };
  const save = makeKeyDefinition(opts);

  function saveEvent() {
    console.log("Saved");
  }
</script>
```
