### keyboardEvents

The `keyboardEvents` used to trigger the events based on the key stroke.

**Available Events:**
`enter`, `tab`, `escape`, `space`, `leftarrow`, `rightarrow`, `downarrow`, `uparrow`, `backspace`, `del`, `shift + {key}`, `ctrl + {key}`

**Usage Example:**

```js
<input use:`event`={customEvent}` />
```

**Example:**

```html
<input type="tex" use:enter={enterEvent} use:tab={tabEvent} />
<script>
  import {
    enter,
    tab,
    escape,
    space,
    leftarrow,
    rightarrow,
    downarrow,
    uparrow,
    backspace,
    del,
    ctrl
  } from "@kws3/ui/keyboard";

  function enterEvent() {
    console.log();("Enter Pressed");
  }
</script>
```
