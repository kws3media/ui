### keyboardEvents

The `keyboardEvents` used to trigger the events based on the key stroke.

**Parameters:**
node : `type: Object`

**Example:**

```js
<input type="tex" use:enter={enterEvent} class="input" />
<input type="tex" use:tab={tabEvent} class="input" />
<script>
  import {
    enter,
    tab,
    escape,
    space,
    leftarrow,
    rightarrow,
    downarrow,
    uparrow,
    backspace,
    del,
  } from "@kws3/ui/keyboard";

  function enterEvent() {
    console.log();("Enter Pressed");
  }
</script>
```
