### debounce

The `Debounce` method Returns a function, that, as long as it continues to be invoked, will not be triggered. The function will be called after it stops being called for `threshold` milliseconds..

**Example:**

```
<input bind:value={keywords} on:keyup={debouncedSearch} />

<script>
  import { debounce } from "@kws3/ui/utils";
  const debouncedSearch = debounce(search, 300);

  function search() {}
</script>
```
