## randomPercent

The `randomPercent` returns random percentage. Default value for min and max are 1 & 100 respectively.

#### Usage

```js
import { randomPercent } from "@kws3/ui/utils";
randomPercent(min, max);
randomPercent(min);
randomPercent();
```
