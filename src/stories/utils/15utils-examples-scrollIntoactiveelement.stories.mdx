### scrollIntoActiveElement

The `scrollIntoActiveElement` used to scroll the list to active element.

**Parameters:**

1. listElement : `type: Object`
2. activeElelement : `type: Object`

**Example:**

```js
<ul bind:this={dropdown}>
  <li>Option 1</li>
  <li>Option 2</li>
  <li class="active">Active Option</li>
</ul>;
import { scrollIntoActiveElement } from "@kws3/ui";

let activeElem = dropdown.querySelector(".active");
scrollIntoActiveElement(dropdown, activeElem);
```
