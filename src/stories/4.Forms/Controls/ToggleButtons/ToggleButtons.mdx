### Usage
#### Import the component:
<Source
  code={
    dedent`import { ToggleButtons  } from '@kws3/ui';`
  }
/>

#### Import the SCSS:
_Default values shown as example_
<Source
  language="css"
  code={
    dedent`
      @use '@kws3/ui/styles/ToggleButtons.scss' with(
        $count-background: cv.getVar("danger"),
        $count-color: cv.getVar("danger-invert"),
      );
    `
  }
/>

#### Custom Active and Inactive classes
Each button can have a custom `active_class` or `inactive_class`.

This class will override the main active/inactive class of the component, **Example:**
<Source
  language="jsx"
  code={
    dedent`
    <ToggleButtons
      ...
      options='{[
        {..., active_class:"is-success", inactive_class:"is-outlined is-success"},
        {..., active_class:"is-danger", inactive_class:"is-outlined is-danger"},
        {..., active_class:"is-info", inactive_class:"is-outlined is-info"}
      ]}'
    />
    `
  }
/>
