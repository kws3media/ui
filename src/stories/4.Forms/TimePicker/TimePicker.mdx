### Usage

#### Import the component:

<Source
  code={dedent`import { Timepicker } from '@kws3/ui';`}
/>

#### Import the SCSS:

<div className="notification is-warning">
  <strong>Note:</strong> <em>Need to import flatpicker SCSS file first</em>
</div>
<Source
  code={dedent`
    @use 'flatpickr/dist/flatpickr';
    @use '@kws3/ui/styles/Timepicker.scss';`}
/>

#### SCSS customization variables with default values:

<Source
  language="css"
  code={dedent`
      @use '@kws3/ui/styles/Timepicker.scss' with(
        $theme-colors: dv.$colors !default;
        $background: cv.getVar("background") !default;
        $text: cv.getVar("text") !default;
      );
    `}
/>
