## Development
 After pulling repo run
 ```bash
 npm run bootstrap
 ```
 it will download all package's dependencies
 then run
  ```bash
 npm run dev:hmr
 ```
 App will run on localhost:9009


## Show
To see packages run
 ```bash
 npm run list
 ```


## Publish
To publish packages run
 ```bash
 npm run publish
 ```

 It will automatically track packages and bump versions by asking from cli
